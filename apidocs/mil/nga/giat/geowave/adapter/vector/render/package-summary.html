Package mil.nga.giat.geowave.adapter.vector.render

  • Interface Summary 
    Interface Description
    DistributableRenderer
    This interface is used to perform rendering within tablet servers.
    WritableRasterCallback
    This interface can be given to an EventingWritableRaster to receive notifications as pixels are written to an AWT WritableRaster.
  • Class Summary 
    Class Description
    DelayedBackbufferGraphic
    A graphic drawing on a BufferedImage compatible with a main graphic.
    EventingWritableRaster
    This class wraps an AWT WritableRaster with a callback to be able to notify when pixels are written for the raster.
    PersistableRenderedImage
    This class wraps a rendered image as a GeoWave Persistable object.
    RenderedMaster
    In addition to a main persisted image, this class also wraps rendered styles.
    RenderedStyle
    This class represents a persisted BufferedImage with a given style ID so that it can be matched to the appropriate style ID for rendering in the correct order in the composite image.
    ServerDecimationOptions
    This class is responsible for persisting any options available to decimation when performing distributed rendering.
    ServerFeatureStyle
    This class persists and encapsulates the rules and tyle information associated with a GeoTools SLD.
    ServerMapArea
    This class persists information associated with the requested map - Coordinate Reference System and bounding box.
    ServerPaintArea
    This class persists information about the image (width, height).
    ServerRenderOptions
    This class persists and encapsulates all of the main rendering information for a layer (any render configuration that is not self-contained within a single style).
