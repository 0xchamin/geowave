    public class MapReduceJobController
extends Object
implements MapReduceJobRunner, IndependentJobRunner
    Run a series of jobs in a sequence. Use the MapReduceJobController.PostOperationTask to allow job definitions to perform an action after running. The purpose of this added task is to support information from a prior job in the sequence(such as temporary file names, job IDs, stats) to be provided to the next job or set of jobs.
