mil.nga.giat.geowave.core.index.simple

Class SimpleNumericIndexStrategy<T extends Number>

  • java.lang.Object
    • mil.nga.giat.geowave.core.index.simple.SimpleNumericIndexStrategy<T>
  • All Implemented Interfaces:
    NumericIndexStrategy, Persistable
    Direct Known Subclasses:
    SimpleIntegerIndexStrategy, SimpleLongIndexStrategy, SimpleShortIndexStrategy

    public abstract class SimpleNumericIndexStrategy<T extends Number>
extends Object
implements NumericIndexStrategy
    A simple 1-dimensional NumericIndexStrategy that represents an index of signed integer values (currently supports 16 bit, 32 bit, and 64 bit integers). The strategy doesn't use any binning. The ids are simply the byte arrays of the value. This index strategy will not perform well for inserting ranges because there will be too much replication of data.
