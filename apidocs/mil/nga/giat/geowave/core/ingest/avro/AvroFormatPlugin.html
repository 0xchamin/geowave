Interface AvroFormatPlugin<I,O>

  • Type Parameters:
    I - The type for the input data
    O - The type that represents each data entry being ingested
    All Superinterfaces:

    All Known Implementing Classes:

    public interface AvroFormatPlugin<I,O>
extends AvroPluginBase<I>, LocalPluginBase, IndexProvider
    This is the main plugin interface for reading from a local file system, and formatting intermediate data (for example, to HDFS or to Kafka for further processing or ingest) from any file that is supported to Avro.
