Interface IngestDimensionalityTypeProviderSpi

  • All Known Implementing Classes:
    SpatialDimensionalityTypeProvider, SpatialTemporalDimensionalityTypeProvider

    public interface IngestDimensionalityTypeProviderSpi
    This interface can be injected using SPI to determine which supported index for an ingest type will be used.

    • Method Detail

      • getCompatibilityVisitor

        IndexCompatibilityVisitor getCompatibilityVisitor()
        get a visitor than can assess the compatibility of an index
        Returns:
        a visitor than can assess the compatibility of any supported index with this dimensionality type. It is best for this visitor to very specifically identify only indices that well match this dimensionality type - for example, if two dimensionality types exist, spatial and spatial-temporal, it is best if the spatial dimensionality type is not over-selective if the existence of a time dimension would better warrant using a spatial-temporal index.

      • getDimensionalityTypeName

        String getDimensionalityTypeName()
        This will represent the name for the dimensionality type that is registered with the ingest framework and presented as a dimensionality type option via the commandline. For consistency, this name is preferably lower-case and without spaces, and should uniquely identify the dimensionality type as much as possible.
