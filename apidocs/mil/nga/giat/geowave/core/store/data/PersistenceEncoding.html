Class PersistenceEncoding<T>

  • java.lang.Object
    • mil.nga.giat.geowave.core.store.data.PersistenceEncoding<T>
  • Direct Known Subclasses:
    IndexedPersistenceEncoding

    public class PersistenceEncoding<T>
extends Object
    This class models all of the necessary information for persisting data in the data store (following the common index model) and is used internally within GeoWave as an intermediary object between the direct storage format and the native data format. It is the responsibility of the data adapter to convert to and from this object and the native object. It does not contain any information about the entry in a particular index and is used when writing an entry, prior to its existence in an index.
