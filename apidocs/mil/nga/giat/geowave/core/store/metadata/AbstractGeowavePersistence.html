mil.nga.giat.geowave.core.store.metadata

Class AbstractGeowavePersistence<T extends Persistable>

  • java.lang.Object
    • mil.nga.giat.geowave.core.store.metadata.AbstractGeowavePersistence<T>
  • Type Parameters:
    T - The type of persistable object that this stores
    Direct Known Subclasses:
    AbstractAccumuloPersistence, AbstractHBasePersistence

    public abstract class AbstractGeowavePersistence<T extends Persistable>
extends Object
    This abstract class does most of the work for storing persistable objects in Geowave datastores and can be easily extended for any object that needs to be persisted. There is an LRU cache associated with it so staying in sync with external updates is not practical - it assumes the objects are not updated often or at all. The objects are stored in their own table.
