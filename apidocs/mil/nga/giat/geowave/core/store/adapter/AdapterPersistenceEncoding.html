    public class AdapterPersistenceEncoding
extends AbstractAdapterPersistenceEncoding
    This is an implementation of persistence encoding that also contains all of the extended data values used to form the native type supported by this adapter. It does not contain any information about the entry in a particular index and is used when writing an entry, prior to its existence in an index.
