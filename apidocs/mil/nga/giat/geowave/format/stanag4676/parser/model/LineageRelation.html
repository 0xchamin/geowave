mil.nga.giat.geowave.format.stanag4676.parser.model

Class LineageRelation

  • java.lang.Object
    • mil.nga.giat.geowave.format.stanag4676.parser.model.LineageRelation

  • public class LineageRelation
extends Object
    Associates related and possibly related tracks to each other. Often there is ambiguity as to whether two tracks are actually the same object. Additionally, multiple objects may converge to appear as a single object or, multiple objects may split from a single track to multiple tracks. The LineageRelation allows all track segments which may be interconnected or related to be identified.
